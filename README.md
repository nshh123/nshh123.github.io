## To_Do-List
*A simple to do list app with HTML, SASS / CSS, and Vanilla Javascript. It is responsive with mobile first in mind. It is built with accessibility (A11y) in mind for keyboard navigation and screen readers. I also used the Web Storage API to create a persistent data store so our list items reappear any time we visit the web app.*

Click this link to use: https://to-do-list-sepia-iota.vercel.app/
